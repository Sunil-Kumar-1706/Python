11.Can variable names start with an underscore (_) or contain special characters like $ or @? Explain the rules.
variable name allowed:
Letters (a–z, A–Z)
Digits (0–9) – but not as the first character
Underscore _ – allowed anywhere and can be the first character
Not Allowed in Variable Names:
Special characters.
Reserved keywords (like class, def, if, while, etc.) cannot be used as variable names.
